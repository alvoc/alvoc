    CLI Reference

    alvoc

    Abundance learning for variants of concern

    Usage:

    alvoc [OPTIONS] COMMAND [ARGS]...

    Options:

    Name Type Description Default
    --version boolean Current version of Alvoc None
    --install-completion boolean Install completion for the current shell. None
    --show-completion boolean Show completion for the current shell, to copy it or customize the installation. None
    --help boolean Show this message and exit. False

    Subcommands

    • amplicons: Get amplicon metrics such as coverage, gc_content and visualizations
    • constellations: Tools to make constellations
    • convert: Tools to convert mutations
    • extract-gene-data: Extracts gene coordinates and the genome sequence from a GenBank file or generates them using an Entrez API search term.
    • find-lineages: Find lineages in samples
    • find-mutants: Find mutations in sequencing data, either from BAM files or a sample list.

    amplicons

    Get amplicon metrics such as coverage, gc_content and visualizations

    Usage:

    alvoc amplicons [OPTIONS] VIRUS SAMPLES INSERTS_PATH

    Options:

    Name Type Description Default
    --outdir, -o text Output directory for results and intermediate data .
    --help boolean Show this message and exit. False

    constellations

    Tools to make constellations

    Usage:

    alvoc constellations [OPTIONS] COMMAND [ARGS]...

    Options:

    Name Type Description Default
    --help boolean Show this message and exit. False

    Subcommands

    • msa: Generates constellations using a custom MSA FASTA file.
    • nextstrain: Generates constellations using the provided nextstrain phylogeny dataset.

    msa

    Generates constellations using a custom MSA FASTA file.

    Usage:

    alvoc constellations msa [OPTIONS] FASTA

    Options:

    Name Type Description Default
    --proportion_threshold, -pt float Minimum proportion of nodes in a clade required to include a mutation 0.9
    --clade-delim, -cd text Delimiter character/string to split each FASTA header on None
    --clade-field, -cf integer Zero-based field index, after splitting on --clade-delim, that contains the clade name None
    --outdir, -o path Output directory for constellations .
    --help boolean Show this message and exit. False

    nextstrain

    Generates constellations using the provided nextstrain phylogeny dataset.

    Usage:

    alvoc constellations nextstrain [OPTIONS] TREE_URL

    Options:

    Name Type Description Default
    --proportion_threshold, -pt float Minimum proportion of nodes in a clade required to include a mutation 0.9
    --use-subclades, -s boolean Use subclades instead of clades for constellation generation False
    --outdir, -o text Output directory for results and intermediate data .
    --help boolean Show this message and exit. False

    convert

    Tools to convert mutations

    Usage:

    alvoc convert [OPTIONS] COMMAND [ARGS]...

    Options:

    Name Type Description Default
    --help boolean Show this message and exit. False

    Subcommands

    • aa: Convert amino acid mutation to nucleotide mutations for a given virus.
    • nt: Convert nucleotide mutation to amino acid mutation for a given virus.

    Convert amino acid mutation to nucleotide mutations for a given virus.

    Options:

    Name Type Description Default
    --outdir, -o text Output directory for results and intermediate data .
    --help boolean Show this message and exit. False

    find-lineages

    Find lineages in samples

    Usage:

    alvoc find-lineages [OPTIONS] VIRUS SAMPLES CONSTELLATIONS

    Options:

    Name Type Description Default
    --white-list, -wl path Path to a TXT file containing lineages to inclue. None
    --black-list, -bl path Path to a TXT file containing lineages to exclude. None
    --min-depth, -d integer Minimum depth for a mutation to be considered. 40
    --unique, -u boolean Whether to consider unique mutations only. False
    --l2, -l2 boolean Whether to use a secondary method for regression analysis. False
    --outdir, -o text Output directory for results and intermediate data .
    --help boolean Show this message and exit. False

    find-mutants

    Find mutations in sequencing data, either from BAM files or a sample list.

    Usage:

    alvoc find-mutants [OPTIONS] VIRUS SAMPLES CONSTELLATIONS MUTATIONS

    Options:

    Name Type Description Default
    --min-depth, -d integer Minimum depth 10
    --outdir, -o text Output directory for results and intermediate data .
    --help boolean Show this message and exit. False
